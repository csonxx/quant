# Lab 03 回测时序和成本

## 目标

理解信号延迟和交易成本如何改变回测。

## 命令

```bash
make demo
python3 -m quant_learning.run_demo --fee-bps 50
```

## 练习

1. 比较 5bp 和 50bp 下的最终权益。
2. 找到 `src/quant_learning/backtest.py` 里计算费用的位置。
3. 找到测试里验证“持仓日期晚于信号日期”的断言。
4. 写一句话解释为什么同日信号同日成交危险。

## 答案方向

如果策略只在低成本假设下赚钱，它的优势可能很薄。信号必须先产生，交易才能发生。
