# Lab 07 机器学习切分

## 目标

在训练模型前，先学会画清楚特征窗口和标签窗口。

## 练习 A：窗口标注

假设今天是 `t`：

```text
特征：过去 5 天收益，使用 t-5 到 t 的数据
标签：未来 5 天收益，使用 t 到 t+5 的数据
```

画成：

```text
t-5 t-4 t-3 t-2 t-1 t | t+1 t+2 t+3 t+4 t+5
feature window          | label window
```

回答：

1. 在 t 时刻，标签是否已经知道？
2. 训练样本和测试样本边界附近为什么会重叠？
3. 哪些样本应该 purge？

## 练习 B：准确率陷阱

假设模型 80% 时间预测上涨正确，但正确时平均赚 0.2%，错误时平均亏 2%。

期望：

```text
0.8 * 0.2% - 0.2 * 2% = -0.24%
```

回答：为什么准确率高仍然亏钱？

## 答案方向

机器学习不是先训练模型，而是先定义不泄漏的预测问题。预测指标必须能转化为交易收益。
