# Lab 06 组合构建

## 目标

理解组合收益来自权重和各资产收益，不是简单把好股票堆在一起。

## 练习 A：等权组合

三只资产当日收益：

```text
A: 2%
B: -1%
C: 0.5%
```

等权组合收益：

```text
(2% - 1% + 0.5%) / 3 = 0.5%
```

## 练习 B：非等权组合

权重：

```text
A: 50%
B: 30%
C: 20%
```

收益：

```text
0.5 * 2% + 0.3 * -1% + 0.2 * 0.5% = 0.8%
```

## 练习 C：集中度审查

假设你选了 10 只股票，其中 8 只是同一行业。

回答：

1. 这算不算真正分散？
2. 如果这个行业突然下跌，组合会怎样？
3. 你会设置什么行业权重上限？

## 答案方向

组合不是数量游戏，而是风险暴露游戏。买得多不等于分散，相关性和集中度才是关键。
