# Lab 08 执行和复盘

## 目标

把研究结果变成纸面交易规则，并写清什么时候暂停或下线。

## 练习 A：纸面交易规则

选一个简单策略，比如：

```text
收盘后快均线高于慢均线，下一交易日买入。
快均线低于慢均线，下一交易日卖出。
```

必须写：

- 数据更新时间。
- 信号生成时间。
- 预计成交时间。
- 成本假设。
- 最大仓位。
- 单笔最大亏损。

## 练习 B：亏损分类

发生一次亏损后，把它归类：

- 策略内亏损。
- 执行问题。
- 数据问题。
- 人工违规。
- 策略失效。

不要一亏就改参数。先分类。

## 练习 C：下线条件

至少写 3 条：

- 最大回撤超过多少暂停。
- 连续亏损多少次降仓。
- 成本超过回测假设多少倍停用。

## 答案方向

没有下线条件的策略，不是策略，是祈祷。复盘的目标不是找借口，而是决定继续、降级、暂停还是杀掉。
