# 检查清单

这些清单用来防止你被漂亮结果骗。

## 策略想法清单

- 这个想法赚的是谁的钱？
- 对手方为什么会长期犯这个错？
- 这条规律是否可能只是牛市 beta？
- 如果反过来做会怎样？
- 是否有一个朴素基准能解释大部分收益？

## 数据清单

- 数据字段定义清楚吗？
- 数据在交易时点已经可知吗？
- 是否存在幸存者偏差？
- 是否处理复权、分红、拆股？
- 是否有停牌、涨跌停、缺失值？
- 是否有重复日期或异常价格？

## 回测清单

- 信号和成交是否错开？
- 是否计算费用、滑点、冲击成本？
- 是否有买入持有基准？
- 是否有样本外测试？
- 是否记录所有失败参数，不只保留成功结果？
- 去掉最赚钱的几笔交易后是否还有效？

## 风控清单

- 单笔最多亏账户多少？
- 组合最多暴露多少？
- 连续亏损几次后降仓？
- 最大回撤到多少暂停？
- 什么条件触发永久下线？

## 因子清单

- 因子是否有经济含义？
- 高分组是否稳定跑赢低分组？
- IC 是否稳定？
- 是否被行业、市值或流动性解释？
- 换手和成本是否杀死收益？
- 容量是否现实？

## 机器学习清单

- 标签定义清楚吗？
- 特征在预测时点可知吗？
- 训练/验证/测试是否按时间切分？
- 标签窗口重叠是否 purge？
- 预测指标是否转化为交易收益？
- 模型是否只是学到了市场 beta？

## 执行清单

- 数据是否准时更新？
- 信号是否准时生成？
- 目标仓位和真实仓位是否一致？
- 订单是否全部成交？
- 成交价是否偏离回测假设？
- 成本是否超过预期？
- 异常是否有暂停机制？
