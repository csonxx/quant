# 最小数学

量化不需要一开始学很深的数学，但这些必须会。

## 百分比

```text
收益率 = 新值 / 旧值 - 1
```

从 100 到 120：

```text
120 / 100 - 1 = 20%
```

从 120 到 100：

```text
100 / 120 - 1 = -16.67%
```

涨 20% 后跌 16.67% 才回本。

## 复利

多期收益连乘：

```text
总收益 = (1 + r1) * (1 + r2) * ... * (1 + rn) - 1
```

不要把收益率简单相加。

## 期望值

```text
期望 = 概率1 * 结果1 + 概率2 * 结果2 + ...
```

交易里常写成：

```text
期望 = 胜率 * 平均盈利 - 败率 * 平均亏损 - 成本
```

## 方差和波动

波动衡量收益围绕平均值上下摆动的程度。

高波动不等于一定亏钱，但高波动会制造大回撤，让人难以执行。

## 相关性

相关性衡量两个资产是否一起动。

- 接近 1：经常同涨同跌。
- 接近 0：关系弱。
- 接近 -1：经常反向。

组合分散依赖低相关，但危机时相关性可能上升。

## 最大回撤

```text
回撤 = 当前权益 / 历史最高权益 - 1
```

最大回撤是所有回撤里的最差值。

## 夏普比率

```text
夏普 = 平均超额收益 / 收益波动
```

它衡量单位波动带来的收益，但不能单独判断策略好坏。夏普可能被短样本、平滑收益、低估成本和尾部风险骗高。
